# Onlyou - Sistema de gestión y tienda online

Proyecto de práctica de desarrollo web.
Sistema para un emprendimiento de lencería con gestión administrativa y vista pública.

## Tecnologías
- C#
- ASP.NET Core
- Blazor WebAssembly
- HTML
- CSS
- JavaScript

## Descripción
Aplicación desarrollada como parte de nuestra formación y práctica profesional.
Incluye gestión de categorías y productos, y una tienda online conectada a un emprendimiento real.
